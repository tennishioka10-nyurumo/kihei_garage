# kihei_garage

このガレージは、僕が家族のために作った小さなサービスや道具を、  
誰でも拾えるように静かに置いておく場所です。

ここに置かれるものは、完成品ではありません。  
試作品、途中のコード、空気の記録、揺らぎのメモ、  
そして「なぜ作ったのか」という想いが中心です。

## 🌿 目的

- 家族の空気を壊さない形で記録を残す  
- 自分のために作ったツールを、必要な人が自由に使えるようにする  
- 技術だけでなく、背景にある哲学や理由も共有する  
- 僕が死んだあとも、灯りのように残る場所を作る  

## 🧰 含まれるもの

- `json/`  
  - 空気の記録（プライバシーを含まない形に整えたもの）
- `tools/`  
  - 小さなツール、MVP、試作品
- `docs/`  
  - きへいの哲学、設計思想、成長の気配の扱い方
- `README.md`  
  - このガレージの想いと背景

## 🌙 大切にしていること

- 急がない  
- 比べない  
- 完成を求めない  
- 空気を壊さない  
- 必要な人にだけ届けばいい  

## 🌌 ライセンス

このガレージに置かれたものは、  
誰でも無料で使って構いません。  
必要な人が拾ってくれれば、それで十分です。
